## Floating label text fields

When the user engages with the text input field, the floating inline labels move to float above the field.

<Example>
    <FormField
        labelFloat
        placeholder="Floating label"
        tips="Help text placed inside Floating label."
    />
    <FormField
        type="datetime-local"
        required
        labelFloat
        label="Date and time"
    />
    <FormField is="select" required labelFloat label="Select field box">
        <option value="" />
        <option>1</option>
        <option>2</option>
        <option>3</option>
    </FormField>
    <FormField is="textarea" labelFloat placeholder="1 row textarea" rows={1} />
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';
import { FormField } from 'material-cell/source/Form/FormField';

render(
    <>
        <FormField
            labelFloat
            placeholder="Floating label"
            tips="Help text placed inside Floating label."
        />
        <FormField
            type="datetime-local"
            required
            labelFloat
            label="Date and time"
        />
        <FormField is="select" required labelFloat label="Select field box">
            <option value="" />
            <option>1</option>
            <option>2</option>
            <option>3</option>
        </FormField>
        <FormField is="textarea" labelFloat placeholder="1 row textarea" rows={1} />
    </>
);
```

## Text field boxes

Text field boxes increase text field identifiability and scannability by using a transparent rectangular fill to enclose the label and input text.

<Example>
    <FormField
        boxMode
        label="Text field box"
        placeholder="Placeholder"
        tips="Help text placed inside Text Field box."
    />
    <FormField is="select" boxMode label="Select field box">
        <option>1</option>
        <option>2</option>
        <option>3</option>
    </FormField>
    <FormField is="select" multiple boxMode label="Multiple select">
        <option>1</option>
        <option>2</option>
        <option>3</option>
    </FormField>
    <FormField is="select" size={3} boxMode label="Size select">
        <option>1</option>
        <option>2</option>

## With icons

Icons describe the type of input a text field requires. They are displayed to the left of the text field.

<Example>
    <InputGroup size="lg">
        <Icon name="event" />
        <Field
            placeholder="Large"
            aria-label="Large icon signifier example input"
        />
    </InputGroup>
    <InputGroup>
        <Icon name="event" />
        <Field
            placeholder="Default"
            aria-label="Default icon signifier example input"
        />
    </InputGroup>
    <InputGroup size="sm">
        <Icon name="event" />
        <Field
            placeholder="Small"
            aria-label="Small icon signifier example input"
        />
    </InputGroup>
    <InputGroup>
        <Icon name="event" />
        <Field
            is="select"
            multiple
            aria-label="Icon signifier with multiple select example"
        >
            <option>With multiple select</option>
            <option>1</option>
            <option>2</option>
            <option>3</option>
        </Field>
    </InputGroup>
    <InputGroup>
        <Icon name="event" />
        <Field
            is="textarea"
            rows="3"
            placeholder="With textarea"
            aria-label="Icon signifier with textarea example"
        />
    </InputGroup>
</Example>

```TSX
import { render, createCell, Fragment } from 'web-cell';

                <option>1</option>
```

## Switches

Switches toggle the state of a single settings option. The option that the switch controls,
as well as the state itâ€™s in, should be made clear from the corresponding inline label.

<Example>
    <ToggleField type="checkbox" switch>
        Toggle this custom switch
    </ToggleField>
</Example>

```TSX
import { render, createCell } from 'web-cell';
import { ToggleField } from 'material-cell/source/Form/ToggleField';

render(
    <ToggleField type="checkbox" switch>
        Toggle this custom switch
    </ToggleField>
);
```
